Colors with keywords as defined by CSS Color Modules at levels 1, 2.1, 3, and 4.

At level 4 was added the rebeccapurple keyword to honor web pioneer Eric A. Meyer whose daughter Rebecca Alison passed away when she was six years, eleven and a half hours old, on Friday June 7 th, 2014.

There are 139 unique colors and 9 synonyms:

  1. aqua = cyan
  2. darkgray = darkgrey
  3. darkslategray = darkslategrey
  4. dimgray = dimgrey
  5. fuchsia = magenta
  6. gray = grey
  7. lightgray = lightgrey
  8. lightslategray = lightslategrey
  9. slategray = slategrey

This table gives information about those colors in the RGB (Red, Green, Blue), HSL (Hue, Saturation, Lightness), and HWB (Hue, Whitness, Blackness) spaces. HWB seems still unsupported after many years since Module Level 4 was introduced, persisting as a draft... A lack which can be addressed by Sass functions for example.

For more on status of CSS Color Modules and details, please visit W3C (in the hope this link is still okay).
